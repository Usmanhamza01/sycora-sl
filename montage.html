🏛️ Export Excel — modèle officiel DGID

L'application reporte automatiquement la fiche R1, la page de garde, le bilan paysage, le compte de résultat et le tableau des flux de trésorerie dans les cellules exactes du modèle officiel DGID intégré à l'application. Aucun fichier à déposer — le résultat est strictement au format DGID.

⚠️ Le modèle DGID est au référentiel SYSCOHADA. Pour une entité SYCEBNL, les rubriques les plus proches sont reportées : vérifiez avant dépôt.
Bilan, compte de résultat, flux de trésorerie et les principales notes annexes (comptes reportés) sont remplis automatiquement dans le formulaire DGID. Vérifiez les notes avant dépôt ; les rubriques détaillées par zone géographique ou par échéance restent à compléter. Aperçu des notes dans l'onglet ci-dessus.

🖨️ Export PDF

Génère un PDF propre (une page par état : bilan actif, bilan passif, compte de résultat, flux de trésorerie) avec l'en-tête de l'entité. Dans la boîte d'impression, choisissez « Enregistrer au format PDF ».

📗 Liasse complète (masque) — états + 35 notes annexes

Génère la liasse complète dans le masque Excel du système : bilan, compte de résultat, TFT et les 35 notes annexes se remplissent et recalculent automatiquement dans Excel (formules du masque), avec la mise en forme d'origine. Le masque SYCEBNL est intégré ; pour un autre masque, déposez-le.
